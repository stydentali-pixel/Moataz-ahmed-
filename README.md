# Smart Telegram Agent Pro+

بوت إدارة محتوى ونشر وجدولة لقنوات تيليجرام مع نظام صلاحيات وطوابير نشر، محسّن للنشر بثقة أعلى.

## الميزات
- Owner / Admin / Editor / Viewer
- إدارة القنوات والقناة الافتراضية
- نشر فوري ونشر مجدول
- الجدولة **بتوقيت مكة المكرمة**
- استعراض المهام والإحصائيات والمهام المجدولة
- إلغاء المهمة المجدولة
- تعديل نص المهمة المجدولة
- إعادة جدولة المهمة
- إشعار داخل البوت عند نجاح النشر أو فشله
- Facebook scaffold جاهز للتوسعة
- فحص صحة الخدمة والاتصال بقاعدة البيانات وRedis
- حماية أفضل للأذونات وإدارة المهام

## التشغيل المحلي
```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```

## Railway
استخدم هذه الأوامر:
- Build Command: `npm install && npm run build`
- Start Command: `npm start`
- قبل أول تشغيل نفّذ: `npx prisma db push`

## المتغيرات البيئية
انسخ `.env.example` إلى `.env` أو استخدم Secrets.

## ملاحظة مهمة عن النوم
الكود يدعم الجدولة والطوابير، لكن **لا يمكنه منع منصة استضافة مجانية من النوم** وحده.
إذا أردت تنفيذ المهام دائمًا في وقتها فشغله على خدمة Always-On مثل Railway أو VPS.
